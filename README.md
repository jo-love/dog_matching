## "Show me my dog website"
자신에게 맞는 강아지를 퀴즈 형식으로 찾아주는 형식의 사이트입니다. 
기획, 디자인, 개발 등의 과정에 모두 참여하였습니다.

### 개발 인원
조연정(FE), 이원철(BE)

### 적용기술 
Front-End: React, styled-component, TypeScript

### 구현기능

#### Main Page
- 사이트를 소개하는 메인 페이지

#### Quiz Page
- 자신과 맞는 강아지를 찾기 위해 문제를 풀어보는 페이지

#### Result page
- 퀴즈결과를 보여주는 페이지
